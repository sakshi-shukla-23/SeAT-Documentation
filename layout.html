Quick Start

There are many ways to install SeAT in all cases you will need a dedicated server as a minimum using the dacker installation you are free to choose any dacker capable host (yes,even windows!).However,if you choose to install SeAT on bare metal,you'll need a linux distribution.We suggest you choos an LTS version of Ubuntu,Debian or CentOS.

Ignoring the details to get SeAT installed quickly you have a few options.All of them will get you SeAT installed but you still have to manually configure an SSO application on the EVE Online Developers Portal.

Method Comments
Docker Follow our docker installation guides.There is the recommendation way to install SeAT as it will handel near everything for you.
Manually You can always install SeAT by following our manual installation guide .At you need is PHP, MariaDB,Redis and a web server and same Linux appearance.

Warning

While installing SeAT-choose one methiod, and stick to it.Otherwise,you'll get SeAT installed multiple times.
